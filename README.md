# Bildstöd

Visual schedule and picture communication with ARASAAC pictograms.
Search 13,000+ ARASAAC pictograms.

GTK4/Adwaita · Python · v0.4.5 · 148 translatable strings

Part of [Autismappar](https://autismappar.se) — free tools for children with autism, language disorders and ADHD.

## Install

### Ubuntu / Debian
```bash
sudo apt install bildstod
```
Repo: [yeager/debian-repo](https://github.com/yeager/debian-repo)

### Fedora
```bash
sudo dnf install bildstod
```
Repo: [yeager/rpm-repo](https://github.com/yeager/rpm-repo)
