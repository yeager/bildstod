# Bildstöd

Visual communication tool with ARASAAC pictogram search and boards.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install bildstod
```

### Fedora/RPM
```bash
sudo dnf install bildstod
```
